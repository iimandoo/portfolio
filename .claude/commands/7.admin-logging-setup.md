포트폴리오의 모든 페이지 접속을 기록하고, 관리자 대시보드(`/admin`)에서 접속 로그, 통계, 분석을 조회할 수 있도록 설정합니다.

⚠️ **사전 조건**: `/sample-page-create`가 먼저 실행되어야 합니다 (페이지 구조가 필요함).

## 사전 체크

아래 파일/폴더 상태를 확인하세요:

1. **프로젝트 구조**
   - `portfolio/src/app/` — 페이지 구조
   - `portfolio/src/actions/` 또는 `api/` — Server Actions 또는 API Routes 위치
   - `portfolio/src/components/` — 컴포넌트 구조

2. **데이터 저장 방식 확인** (선택해야 함)
   - 옵션 A: **SQLite (간단함, 로컬 저장)**
     - 장점: 배포 간단, 외부 의존성 없음
     - 단점: 서버리스 환경에선 제약
   - 옵션 B: **PostgreSQL (권장, 프로덕션)**
     - 장점: 확장성, 다중 환경 지원
     - 단점: 외부 DB 필요
   - 옵션 C: **In-Memory (개발용, 재시작 시 초기화)**
     - 장점: 설정 최소, 빠른 응답
     - 단점: 데이터 휘발성

3. **기존 DB 설정 여부**
   - `.env.local` 또는 `schema.prisma` 확인

## Architecture

```
포트폴리오 요청
    ↓
middleware.ts (페이지 접속 감지)
    ↓
logPageView Server Action (접속 정보 수집)
    ↓
Database (SQLite/PostgreSQL/Memory)
    ↓
/admin 대시보드 (조회 API)
    ↓
Admin UI (Chart, Table, Filter)
```

#### 1-2. In-Memory 사용 시 (개발용, 빠른 설정)

`portfolio/src/lib/memory-db.ts` 생성:
```typescript
interface PageLog {
  id: number;
  url: string;
  path: string;
  referrer?: string;
  userAgent?: string;
  ipAddress?: string;
  timestamp: Date;
}

let logs: PageLog[] = [];
let nextId = 1;

export const memoryDB = {
  addLog: (data: Omit<PageLog, 'id' | 'timestamp'>) => {
    const log: PageLog = {
      id: nextId++,
      ...data,
      timestamp: new Date(),
    };
    logs.push(log);
    return log;
  },

  getLogs: (limit: number = 100, offset: number = 0) => {
    return logs
      .sort((a, b) => b.timestamp.getTime() - a.timestamp.getTime())
      .slice(offset, offset + limit);
  },

  getStats: () => {
    const stats: Record<string, { count: number; lastVisit: Date }> = {};

    logs.forEach((log) => {
      if (!stats[log.path]) {
        stats[log.path] = { count: 0, lastVisit: log.timestamp };
      }
      stats[log.path].count++;
      if (log.timestamp > stats[log.path].lastVisit) {
        stats[log.path].lastVisit = log.timestamp;
      }
    });

    return stats;
  },

  clearLogs: () => {
    logs = [];
    nextId = 1;
  },
};
```

### 2. Middleware를 통한 접속 감지

`portfolio/src/middleware.ts` 생성 (또는 기존 파일 수정):

```typescript
import { NextRequest, NextResponse } from 'next/server';

export function middleware(request: NextRequest) {
  // /admin 경로는 로깅 스킵
  if (request.nextUrl.pathname.startsWith('/admin')) {
    return NextResponse.next();
  }

  // 정적 자산 제외
  if (
    request.nextUrl.pathname.startsWith('/_next') ||
    request.nextUrl.pathname.startsWith('/api') ||
    /\.(jpg|jpeg|png|gif|ico|svg|css|js)$/.test(request.nextUrl.pathname)
  ) {
    return NextResponse.next();
  }

  // 요청 정보 수집
  const headers = request.headers;
  const userAgent = headers.get('user-agent') || '';
  const referrer = headers.get('referer') || '';
  const ipAddress = headers.get('x-forwarded-for') || headers.get('x-real-ip') || '';

  // 쿠키에 요청 정보 저장 (Server Action으로 전달하기 위함)
  const response = NextResponse.next();
  response.cookies.set('_pageViewData', JSON.stringify({
    url: request.url,
    path: request.nextUrl.pathname,
    referrer,
    userAgent,
    ipAddress,
  }), { maxAge: 1 }); // 1초 유효 (Server Action에서만 사용)

  return response;
}

export const config = {
  matcher: [
    /*
     * 다음 경로를 제외한 모든 요청 매칭:
     * - api (API routes)
     * - _next/static (static files)
     * - _next/image (image optimization files)
     * - favicon.ico (favicon file)
     */
    '/((?!api|_next/static|_next/image|favicon.ico).*)',
  ],
};
```

### 3. Server Action으로 로그 기록

`portfolio/src/actions/logging.ts` 생성:

```typescript
'use server';

import { cookies } from 'next/headers';
import { memoryDB } from '@/lib/memory-db';
// import db from '@/lib/db'; // SQLite 사용 시

export async function logPageView() {
  try {
    const cookieStore = await cookies();
    const pageViewData = cookieStore.get('_pageViewData')?.value;

    if (!pageViewData) return;

    const data = JSON.parse(pageViewData);

    // In-Memory DB 저장
    memoryDB.addLog({
      url: data.url,
      path: data.path,
      referrer: data.referrer,
      userAgent: data.userAgent,
      ipAddress: data.ipAddress,
    });

    // SQLite 저장 시 (주석 처리된 부분 활성화)

    //   VALUES (?, ?, ?, ?, ?)
    // `).run(data.url, data.path, data.referrer, data.userAgent, data.ipAddress);

  } catch (error) {
    console.error('Failed to log page view:', error);
  }
}
```

#### 5-2. Admin 페이지

`portfolio/src/app/admin/page.tsx` 생성:

```typescript
'use client';

import { useEffect, useState } from 'react';
import { Button } from '@/components/ui/button';
import { Card } from '@/components/ui/card';

interface PageLog {
  id: number;
  url: string;
  path: string;
  referrer?: string;
  timestamp: string;
}

interface PageStats {
  [path: string]: {
    count: number;
    lastVisit: string;
  };
}

export default function AdminPage() {
  const [logs, setLogs] = useState<PageLog[]>([]);
  const [stats, setStats] = useState<PageStats>({});
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetchData();
    // 5초마다 갱신
    const interval = setInterval(fetchData, 5000);
    return () => clearInterval(interval);
  }, []);

  const fetchData = async () => {
    try {
      const response = await fetch('/api/admin/logs');
      const data = await response.json();
      setLogs(data.logs || []);
      setStats(data.stats || {});
    } catch (error) {
      console.error('Failed to fetch logs:', error);
    } finally {
      setLoading(false);
    }
  };

  return (
    <div className="space-y-8">
      {/* Summary Cards */}
      <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
        <Card className="p-6">
          <h3 className="text-sm font-medium text-gray-500">Total Views</h3>
          <p className="text-3xl font-bold mt-2">{logs.length}</p>
        </Card>
        <Card className="p-6">
          <h3 className="text-sm font-medium text-gray-500">Unique Pages</h3>
          <p className="text-3xl font-bold mt-2">{Object.keys(stats).length}</p>
        </Card>
        <Card className="p-6">
          <h3 className="text-sm font-medium text-gray-500">Most Popular</h3>
          <p className="text-lg font-semibold mt-2">
            {Object.entries(stats).sort(([, a], [, b]) => b.count - a.count)[0]?.[0] || '-'}
          </p>
        </Card>
      </div>

      {/* Page Statistics Table */}
      <Card>
        <div className="p-6">
          <h2 className="text-lg font-semibold mb-4">Page Statistics</h2>
          {loading ? (
            <p>Loading...</p>
          ) : (
            <table className="w-full text-sm">
              <thead className="border-b">
                <tr>
                  <th className="text-left py-2">Page</th>
                  <th className="text-right py-2">Views</th>
                  <th className="text-right py-2">Last Visited</th>
                </tr>
              </thead>
              <tbody>
                {Object.entries(stats)
                  .sort(([, a], [, b]) => b.count - a.count)
                  .map(([path, stat]) => (
                    <tr key={path} className="border-b">
                      <td className="py-3 font-mono text-xs">{path}</td>
                      <td className="text-right py-3 font-semibold">{stat.count}</td>
                      <td className="text-right py-3 text-gray-500 text-xs">
                        {new Date(stat.lastVisit).toLocaleString()}
                      </td>
                    </tr>
                  ))}
              </tbody>
            </table>
          )}
        </div>
      </Card>

      {/* Recent Logs */}
      <Card>
        <div className="p-6">
          <div className="flex justify-between items-center mb-4">
            <h2 className="text-lg font-semibold">Recent Logs</h2>
            <Button
              variant="outline"
              size="sm"
              onClick={() => {
                fetch('/api/admin/logs/clear', { method: 'POST' })
                  .then(() => fetchData())
                  .catch(console.error);
              }}
            >
              Clear Logs
            </Button>
          </div>
          {loading ? (
            <p>Loading...</p>
          ) : (
            <div className="space-y-2 max-h-96 overflow-y-auto">
              {logs.slice(0, 50).map((log) => (
                <div key={log.id} className="flex flex-col text-xs p-2 bg-gray-50 rounded">
                  <div className="font-mono text-gray-700">{log.path}</div>
                  <div className="text-gray-500">
                    {new Date(log.timestamp).toLocaleString()}
                  </div>
                </div>
              ))}
            </div>
          )}
        </div>
      </Card>
    </div>
  );
}
```

### 6. Admin API Routes

`portfolio/src/app/api/admin/logs/route.ts` 생성:

```typescript
import { NextResponse } from 'next/server';
import { memoryDB } from '@/lib/memory-db';

export async function GET() {
  const logs = memoryDB.getLogs(100, 0);
  const stats = memoryDB.getStats();

  return NextResponse.json({
    logs: logs.map((log) => ({
      ...log,
      timestamp: log.timestamp.toISOString(),
    })),
    stats: Object.entries(stats).reduce(
      (acc, [path, data]) => {
        acc[path] = {
          count: data.count,
          lastVisit: data.lastVisit.toISOString(),
        };
        return acc;
      },
      {} as Record<string, any>
    ),
  });
}
```

## Rules

- **Admin 접근 제어**: 현재는 오픈 상태. 프로덕션에서는 인증 추가 필수 (예: Basic Auth, API Key)
- **성능**: 로그는 쓰기만 수행하고, 읽기는 최소화 (캐싱 권장)
- **데이터 보존**: In-Memory는 서버 재시작 시 초기화됨 (로컬 개발용)
- **GDPR/개인정보**: IP 수집 시 개인정보 정책 업데이트 필요
- **정적 자산**: JS, CSS, 이미지 등은 로깅 대상에서 제외
