포트폴리오의 검색 엔진 최적화(SEO)를 설정합니다.
동적 메타데이터, 구조화된 데이터(Schema.org), sitemap, robots.txt 등을 자동으로 생성합니다.

⚠️ **사전 조건**: `/content-setup` 명령어가 먼저 실행되어야 합니다 (메타데이터 필요).

## 목표

- 모든 페이지에 동적 메타데이터 적용
- Open Graph (OG) 메타데이터 설정
- 구조화된 데이터 (Schema.org)
- sitemap.xml 자동 생성
- robots.txt 설정
- Core Web Vitals 최적화
- 검색 엔진 최적화

## Steps

**`src/app/projects/[id]/page.tsx`** (프로젝트 상세 페이지):

```typescript
import type { Metadata } from 'next';
import { getProjects, getMetadata } from '@/lib/data-loader';

const meta = getMetadata();
const projects = getProjects();

interface Props {
  params: { id: string };
}

export function generateMetadata({ params }: Props): Metadata {
  const project = projects.projects.find((p) => p.id === params.id);

  if (!project) {
    return { title: '프로젝트 없음' };
  }

  return {
    title: project.title,
    description: project.description,
    openGraph: {
      title: project.title + ' | ' + meta.siteTitle,
      description: project.description,
      images: project.image ? [{ url: project.image }] : [],
    },
  };
}

export function generateStaticParams() {
  return projects.projects.map((project) => ({
    id: project.id,
  }));
}

export default function ProjectPage({ params }: Props) {
  const project = projects.projects.find((p) => p.id === params.id);

  if (!project) {
    return <div>프로젝트를 찾을 수 없습니다</div>;
  }

  return (
    // ... 프로젝트 상세 페이지
  );
}
```

### 4. sitemap.xml 동적 생성

**`src/app/sitemap.ts`** 생성:

```typescript
import type { MetadataRoute } from 'next';
import { getProjects, getMetadata } from '@/lib/data-loader';

const meta = getMetadata();
const projects = getProjects();

export default function sitemap(): MetadataRoute.Sitemap {
  const projectEntries = projects.projects.map((project) => ({
    url: `${meta.siteUrl}/projects/${project.id}`,
    lastModified: new Date(project.period.end),
    changeFrequency: 'monthly' as const,
    priority: 0.7,
  }));

  return [
    {
      url: meta.siteUrl,
      lastModified: new Date(),
      changeFrequency: 'weekly',
      priority: 1,
    },
    {
      url: `${meta.siteUrl}/projects`,
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 0.8,
    },
    ...projectEntries,
    {
      url: `${meta.siteUrl}/resume`,
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 0.8,
    },
    {
      url: `${meta.siteUrl}/about`,
      lastModified: new Date(),
      changeFrequency: 'monthly',
      priority: 0.8,
    },
  ];
}
```

### 5. RSS Feed 생성 (선택)

**`src/app/feed.xml/route.ts`** 생성:

```typescript
import { getProjects, getMetadata } from '@/lib/data-loader';
import type { NextResponse } from 'next/server';

const meta = getMetadata();
const projects = getProjects();

export async function GET(): Promise<NextResponse> {
  const rss = `<?xml version="1.0" encoding="UTF-8"?>
<rss version="2.0" xmlns:content="http://purl.org/rss/1.0/modules/content/">
  <channel>
    <title>${meta.siteTitle}</title>
    <link>${meta.siteUrl}</link>
    <description>${meta.siteDescription}</description>
    <language>ko</language>
    ${projects.projects
      .map(
        (project) => `
    <item>
      <title>${project.title}</title>
      <link>${meta.siteUrl}/projects/${project.id}</link>
      <description>${project.description}</description>
      <pubDate>${new Date(project.period.start).toUTCString()}</pubDate>
    </item>
    `
      )
      .join('')}
  </channel>
</rss>`;

  return new Response(rss, {
    headers: {
      'Content-Type': 'application/xml',
      'Cache-Control': 'public, s-maxage=3600, stale-while-revalidate=86400',
    },
  });
}
```

## Rules

- **메타데이터**: 모든 페이지에 title, description 필수
- **이미지**: og:image는 최소 1200x630px
- **Canonical**: URL 정규화 필수
- **모바일**: 반응형 디자인 필수 (모바일-first)
- **성능**: Core Web Vitals 최적화 필수
  - LCP: < 2.5초
  - FID: < 100ms
  - CLS: < 0.1
