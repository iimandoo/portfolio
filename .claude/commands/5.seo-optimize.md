🔵 **Phase 2 — 배포 & 운영 최적화** (Phase 1 완료 후 진행)

포트폴리오의 검색 엔진 최적화(SEO)를 설정합니다.
동적 메타데이터, 구조화된 데이터(Schema.org), sitemap, robots.txt 등을 자동으로 생성합니다.

⚠️ **사전 조건**: Phase 1(`/project-init` → `/theme-on` → `/content-setup` → `/sample-page-create`) 완료 후 실행.

## 목표

- 모든 페이지에 동적 메타데이터 적용
- Open Graph (OG) 메타데이터 설정
- 구조화된 데이터 (Schema.org)
- sitemap.xml 자동 생성
- robots.txt 설정
- Core Web Vitals 최적화
- 검색 엔진 최적화

## Steps

### 1. Next.js 메타데이터 API 설정

**`src/app/layout.tsx`** 업데이트:

```typescript
import type { Metadata } from 'next';
import { StyleProvider } from '@/styles/provider';
import '@/styles/globals.css';
import { resume } from '@/data/resume';

const { meta } = resume;

export const metadata: Metadata = {
  metadataBase: new URL(meta.siteUrl),
  title: {
    default: meta.siteTitle,
    template: '%s | ' + meta.siteTitle.split('|')[0].trim(),
  },
  description: meta.siteDescription,
  keywords: meta.keywords,
  authors: [{ name: meta.author }],
  creator: meta.author,

  // Open Graph
  openGraph: {
    type: 'website',
    locale: 'ko_KR',
    url: meta.siteUrl,
    title: meta.siteTitle,
    description: meta.siteDescription,
    images: [
      {
        url: meta.ogImage,
        width: 1200,
        height: 630,
        alt: meta.siteTitle,
        type: 'image/jpeg',
      },
    ],
  },

  // Twitter
  twitter: {
    card: 'summary_large_image',
    title: meta.siteTitle,
    description: meta.siteDescription,
    images: [meta.ogImage],
    creator: '@yourusername', // Twitter 계정 수정
  },

  // Canonical
  canonical: meta.siteUrl,

  // Robots
  robots: {
    index: true,
    follow: true,
    'max-image-preview': 'large',
    'max-snippet': -1,
    'max-video-preview': -1,
    googleBot: 'index, follow',
  },

  // Verification (Google Search Console, Naver Search Advisor)
  verification: {
    google: 'YOUR_GOOGLE_VERIFICATION_CODE', // Google Search Console에서 발급받은 코드
    // naver: 'YOUR_NAVER_VERIFICATION_CODE', // Naver Search Advisor 코드 (선택)
  },
};

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="ko">
      <head>
        {/* Pretendard 폰트 */}
        <link
          href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.css"
          rel="stylesheet"
        />

        {/* Google Fonts Icon */}
        <link
          rel="stylesheet"
          href="https://fonts.googleapis.com/icon?family=Material+Icons"
        />

        {/* Favicon */}
        <link rel="icon" href="/favicon.ico" sizes="any" />

        {/* Preconnect to external services (성능 최적화) */}
        <link rel="preconnect" href="https://fonts.googleapis.com" />
        <link rel="preconnect" href="https://fonts.gstatic.com" crossOrigin="anonymous" />
      </head>
      <body>
        <StyleProvider>
          {children}

          {/* Schema.org 구조화된 데이터 */}
        <script
          type="application/ld+json"
          dangerouslySetInnerHTML={{
            __html: JSON.stringify({
              '@context': 'https://schema.org',
              '@type': 'Person',
              name: resume.profile.name,
              url: meta.siteUrl,
              image: resume.profile.image,
              jobTitle: resume.profile.title,
              sameAs: [
                resume.profile.social.github,
                resume.profile.social.linkedin,
              ].filter(Boolean),
            }),
          }}
        />
        </StyleProvider>
      </body>
    </html>
  );
}
```

### 4. sitemap.xml 동적 생성

**`src/app/sitemap.ts`** 생성:

```typescript
import type { MetadataRoute } from "next";
import { resume } from "@/data/resume";

const { meta } = resume;

// 단일 페이지 구조 — / 경로 하나만 포함
export default function sitemap(): MetadataRoute.Sitemap {
  return [
    {
      url: meta.siteUrl,
      lastModified: new Date(),
      changeFrequency: "weekly" as const,
      priority: 1,
    },
  ];
}
```

### 5. RSS Feed 생성 (선택)

**`src/app/feed.xml/route.ts`** 생성:

```typescript
import { resume } from "@/data/resume";

const { meta, Project } = resume;

export async function GET() {
  const rss = `<?xml version="1.0" encoding="UTF-8"?>
<rss version="2.0" xmlns:content="http://purl.org/rss/1.0/modules/content/">
  <channel>
    <title>${meta.siteTitle}</title>
    <link>${meta.siteUrl}</link>
    <description>${meta.siteDescription}</description>
    <language>ko</language>
    ${Project.cases
      .map(
        (c) => `
    <item>
      <title>${c.title}</title>
      <link>${meta.siteUrl}#Project</link>
      <description>${c.Project}</description>
    </item>
    `,
      )
      .join("")}
  </channel>
</rss>`;

  return new Response(rss, {
    headers: {
      "Content-Type": "application/xml",
      "Cache-Control": "public, s-maxage=3600, stale-while-revalidate=86400",
    },
  });
}
```

### 6. next.config.ts 성능 최적화

**`next.config.ts`** 업데이트:

```typescript
import type { NextConfig } from "next";

const nextConfig: NextConfig = {
  compiler: {
    styledComponents: true,
  },

  // 이미지 최적화
  images: {
    formats: ["image/webp", "image/avif"],
    deviceSizes: [640, 750, 828, 1080, 1200, 1920, 2048, 3840],
    imageSizes: [16, 32, 48, 64, 96, 128, 256, 384],
  },

  // 성능 최적화
  compress: true,
  poweredByHeader: false,
  productionBrowserSourceMaps: false,

  // Headers 설정
  async headers() {
    return [
      {
        source: "/images/:path*",
        headers: [
          {
            key: "Cache-Control",
            value: "public, max-age=31536000, immutable",
          },
        ],
      },
      {
        source: "/fonts/:path*",
        headers: [
          {
            key: "Cache-Control",
            value: "public, max-age=31536000, immutable",
          },
        ],
      },
    ];
  },
};

export default nextConfig;
```

## Rules

- **메타데이터**: 모든 페이지에 title, description 필수
- **이미지**: og:image는 최소 1200x630px
- **Canonical**: URL 정규화 필수
- **모바일**: 반응형 디자인 필수 (모바일-first)
- **성능**: Core Web Vitals 최적화 필수
  - LCP: < 2.5초
  - FID: < 100ms
  - CLS: < 0.1
