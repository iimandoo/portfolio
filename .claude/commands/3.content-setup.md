포트폴리오에 표시될 경력, 프로젝트, 기술 스택 등의 콘텐츠 데이터 구조를 정의하고 샘플 데이터를 생성합니다.

⚠️ **사전 조건**: `/project-init` 명령어가 먼저 실행되어야 합니다.

## 목표

- `src/data/` 폴더에 JSON 스키마 정의
- 사용자 정보, 경력, 프로젝트, 기술 스택 등 구조화된 데이터
- 쉽게 수정할 수 있는 포맷 (JSON → 사용자가 직접 수정 가능)
- Markdown 지원 (프로젝트 상세 설명 등)

## Steps

### 1. 데이터 디렉토리 구조 생성

```
src/data/
├── profile.json          # 사용자 기본 정보
├── career.json           # 경력 데이터 (회사, 기간, 역할)
├── projects.json         # 프로젝트 목록
├── skills.json           # 기술 스택
└── metadata.json         # SEO 메타데이터
```

### 3. TypeScript 타입 정의 (선택)

**`src/types/index.ts`** 생성:

```typescript
export interface Profile {
  name: string;
  title: string;
  subtitle: string;
  email: string;
  phone: string;
  location: string;
  bio: string;
  image: string;
  social: Record<string, string>;
  availability: {
    status: 'open' | 'closed';
    message: string;
  };
}

export interface Experience {
  id: string;
  company: string;
  position: string;
  period: {
    start: string;
    end: string;
  };
  description: string;
  highlights: string[];
  skills: string[];
}

export interface Project {
  id: string;
  title: string;
  company: string;
  description: string;
  role: string;
  period: {
    start: string;
    end: string;
  };
  impact: string[];
  tech: string[];
  link?: string;
  github?: string;
  image?: string;
  caseStudy?: string;
}

export interface Skill {
  name: string;
  level: 'beginner' | 'intermediate' | 'advanced' | 'expert';
  years: number;
}

export interface SkillCategory {
  name: string;
  skills: Skill[];
}

export interface Metadata {
  siteTitle: string;
  siteDescription: string;
  siteUrl: string;
  ogImage: string;
  socialLinks: Record<string, string>;
  author: string;
  keywords: string[];
  theme: {
    primaryColor: string;
    accentColor: string;
  };
}
```

### 4. 데이터 로드 유틸리티 생성

**`src/lib/data-loader.ts`** 생성:

```typescript
import profile from '@/data/profile.json';
import career from '@/data/career.json';
import projects from '@/data/projects.json';
import skills from '@/data/skills.json';
import metadata from '@/data/metadata.json';

export const DATA = {
  profile,
  career,
  projects,
  skills,
  metadata,
};

export const getProfile = () => profile;
export const getCareer = () => career;
export const getProjects = () => projects;
export const getSkills = () => skills;
export const getMetadata = () => metadata;
```

## Rules

- **JSON 구조**: 정의된 스키마를 정확히 따를 것 (TypeScript 타입 참고)
- **이미지 경로**: 모두 `public/images/` 상대 경로 사용
- **기간 형식**: "YYYY-MM" 형식 통일
- **데이터 수정**: JSON 파일을 직접 수정하여 쉽게 업데이트 가능
- **타입 안정성**: TypeScript 타입을 사용하여 컴파일 타임 검증
