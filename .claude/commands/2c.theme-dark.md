🟢 **Phase 1 — 디자인 시스템 · Dark 테마**

> 📌 공용 인프라(`provider.tsx`, `styled.d.ts`, `globals.css`), 공통 Rules, 빌드 확인, 다음 단계는 **`2.theme-on.md`** 참조

## Dark 테마 특징

| 항목 | 값 | 비고 |
|------|----|------|
| 배경 | `#0F1117` | 깊은 다크 네이비 |
| 전경 | `#E8E8E8` | 부드러운 흰색 |
| 강조색 | `#7C3AED` | 퍼플 (또는 `$ARGUMENTS`로 커스텀) |
| 모서리 | `12px` | Toss와 동일한 곡선감 |
| 그림자 | Purple glow | `rgba(124,58,237,0.2)` 글로우 |
| 분위기 | 개발자/테크 포트폴리오 | 미묘한 글로우 효과 |

## 생성할 파일

| 파일 | 역할 |
|------|------|
| `src/styles/themes/dark.ts` | Dark 디자인 토큰 |
| `src/app/dark/layout.tsx` | Dark ThemeProvider 래퍼 |

## Steps

### Step 1: `src/styles/themes/dark.ts` 생성

```typescript
export const darkTheme = {
  colors: {
    background: '#0F1117',
    foreground: '#E8E8E8',
    primary: '#7C3AED',           // 퍼플 강조색 (또는 $ARGUMENTS로 커스텀)
    primaryForeground: '#FFFFFF',
    secondary: '#1A1B26',
    secondaryForeground: '#B0B0C0',
    muted: '#161820',
    mutedForeground: '#6B7280',
    border: '#2A2B38',
    card: '#1A1B26',
    cardForeground: '#E8E8E8',
    destructive: '#FF4444',
  },

  radius: {
    xs: '0.25rem',
    sm: '0.5rem',
    md: '0.75rem',   // Toss와 동일 — 일관된 곡선감
    lg: '1rem',
    xl: '1.5rem',
  },

  shadows: {
    sm: '0 1px 4px rgba(0,0,0,0.4)',
    md: '0 4px 16px rgba(0,0,0,0.5)',
    lg: '0 8px 32px rgba(124,58,237,0.2)',   // 퍼플 글로우
  },

  fonts: {
    primary: "'Pretendard Variable', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif",
  },

  typography: {
    h1: { fontSize: '2.5rem', fontWeight: 700, letterSpacing: '-0.02em' },
    h2: { fontSize: '2rem', fontWeight: 600, letterSpacing: '-0.01em' },
    body: { fontSize: '1rem', fontWeight: 400, lineHeight: 1.6 },
    small: { fontSize: '0.875rem', fontWeight: 400 },
  },
} as const;

export type DarkTheme = typeof darkTheme;
```

### Step 2: `src/app/dark/layout.tsx` 생성

```typescript
'use client';

import { StyleProvider } from '@/styles/provider';
import { darkTheme } from '@/styles/themes/dark';

export default function DarkLayout({ children }: { children: React.ReactNode }) {
  return (
    <StyleProvider theme={darkTheme}>
      <div style={{ minHeight: '100vh', backgroundColor: '#0F1117', color: '#E8E8E8' }}>
        {children}
      </div>
    </StyleProvider>
  );
}
```

## 섹션 컴포넌트 스타일 가이드 (Dark 테마)

`/sample-page-create` 실행 시 참고할 Dark 특화 스타일 지침:

**분위기 원칙:**
- 배경색을 `props.theme.colors.background` (#0F1117)로 통일
- 카드 배경은 `props.theme.colors.card` (#1A1B26) — 배경보다 살짝 밝게
- 강조 요소에만 primary (#7C3AED) 사용 (버튼, 링크, 호버 상태)
- border는 `props.theme.colors.border` (#2A2B38) 로 미묘하게

**컴포넌트 예시:**

```typescript
const HeroTitle = styled.h1`
  font-size: 2.5rem;
  font-weight: 700;
  color: ${(props) => props.theme.colors.foreground};
  letter-spacing: -0.02em;

  /* 강조 단어에 그라디언트 효과 */
  span.highlight {
    background: linear-gradient(135deg, #7C3AED, #A855F7);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
    background-clip: text;
  }
`;

const ProjectCard = styled.div`
  background-color: ${(props) => props.theme.colors.card};
  border: 1px solid ${(props) => props.theme.colors.border};
  border-radius: ${(props) => props.theme.radius.xl};
  transition: box-shadow 0.2s ease, border-color 0.2s ease;

  &:hover {
    box-shadow: ${(props) => props.theme.shadows.lg};
    border-color: ${(props) => props.theme.colors.primary};
  }
`;

const PrimaryButton = styled.button`
  background-color: ${(props) => props.theme.colors.primary};
  color: ${(props) => props.theme.colors.primaryForeground};
  padding: 0.75rem 1.5rem;
  border-radius: ${(props) => props.theme.radius.md};
  font-weight: 600;
  transition: all 0.2s ease;

  &:hover {
    opacity: 0.85;
    box-shadow: 0 0 20px rgba(124, 58, 237, 0.4);
    transform: translateY(-1px);
  }
`;
```
