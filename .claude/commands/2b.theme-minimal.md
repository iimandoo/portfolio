🟢 **Phase 1 — 디자인 시스템 · Minimal 테마**

> 📌 공용 인프라(`provider.tsx`, `styled.d.ts`, `globals.css`), 공통 Rules, 빌드 확인, 다음 단계는 **`2.theme-on.md`** 참조

## Minimal 테마 특징

| 항목 | 값 | 비고 |
|------|----|------|
| 배경 | `#FFFFFF` | 순수 흰색 |
| 전경 | `#000000` | 순수 검정 |
| 강조색 | `#000000` | 색 없이 무게로 강조 |
| 모서리 | `2–6px` | 거의 각진 모서리 |
| 타이포 | Light weight | 타이포그래피 중심 레이아웃 |
| 그림자 | 극도로 미세 | 입체감 최소화 |

## 생성할 파일

| 파일 | 역할 |
|------|------|
| `src/styles/themes/minimal.ts` | Minimal 디자인 토큰 |
| `src/app/minimal/layout.tsx` | Minimal ThemeProvider 래퍼 |

## Steps

### Step 1: `src/styles/themes/minimal.ts` 생성

```typescript
export const minimalTheme = {
  colors: {
    background: '#FFFFFF',
    foreground: '#000000',
    primary: '#000000',           // 강조색 없이 검정으로 통일 (또는 $ARGUMENTS로 커스텀)
    primaryForeground: '#FFFFFF',
    secondary: '#F5F5F5',
    secondaryForeground: '#333333',
    muted: '#FAFAFA',
    mutedForeground: '#888888',
    border: '#E0E0E0',
    card: '#FFFFFF',
    cardForeground: '#000000',
    destructive: '#CC0000',
  },

  radius: {
    xs: '0',
    sm: '2px',
    md: '4px',    // 거의 각진 모서리
    lg: '6px',
    xl: '8px',
  },

  shadows: {
    sm: '0 1px 2px rgba(0,0,0,0.04)',
    md: '0 2px 8px rgba(0,0,0,0.06)',
    lg: '0 4px 16px rgba(0,0,0,0.08)',
  },

  fonts: {
    primary: "'Pretendard Variable', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif",
  },

  typography: {
    h1: { fontSize: '3rem', fontWeight: 300, letterSpacing: '-0.03em' },   // 가벼운 weight
    h2: { fontSize: '2rem', fontWeight: 300, letterSpacing: '-0.02em' },
    body: { fontSize: '1rem', fontWeight: 400, lineHeight: 1.75 },         // 넉넉한 행간
    small: { fontSize: '0.875rem', fontWeight: 400 },
  },
} as const;

export type MinimalTheme = typeof minimalTheme;
```

### Step 2: `src/app/minimal/layout.tsx` 생성

```typescript
'use client';

import { StyleProvider } from '@/styles/provider';
import { minimalTheme } from '@/styles/themes/minimal';

export default function MinimalLayout({ children }: { children: React.ReactNode }) {
  return <StyleProvider theme={minimalTheme}>{children}</StyleProvider>;
}
```

## 섹션 컴포넌트 스타일 가이드 (Minimal 테마)

`/sample-page-create` 실행 시 참고할 Minimal 특화 스타일 지침:

**레이아웃 원칙:**
- 좌우 패딩을 충분히 (최대 너비 `48rem`, 중앙 정렬)
- 섹션 간 여백을 크게 (`padding: 6rem 2rem`)
- 텍스트 계층 구조로 정보를 전달 (컬러 대신 크기/weight 차이)

**컴포넌트 예시:**

```typescript
const SectionTitle = styled.h2`
  font-size: 2rem;
  font-weight: 300;
  letter-spacing: -0.02em;
  color: ${(props) => props.theme.colors.foreground};
  border-bottom: 1px solid ${(props) => props.theme.colors.border};
  padding-bottom: 1rem;
  margin-bottom: 3rem;
`;

const ProjectCard = styled.div`
  border: 1px solid ${(props) => props.theme.colors.border};
  border-radius: ${(props) => props.theme.radius.md};
  padding: 2rem;
  transition: border-color 0.2s ease;

  &:hover {
    border-color: ${(props) => props.theme.colors.foreground};
  }
`;

const Tag = styled.span`
  display: inline-block;
  border: 1px solid ${(props) => props.theme.colors.border};
  border-radius: ${(props) => props.theme.radius.sm};
  padding: 0.2rem 0.6rem;
  font-size: 0.75rem;
  letter-spacing: 0.05em;
  text-transform: uppercase;
  color: ${(props) => props.theme.colors.mutedForeground};
`;
```
