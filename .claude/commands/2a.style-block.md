🟢 **Phase 1 — 디자인 스타일 · Block**

Block 스타일은 전체 너비 카드 블록을 수직으로 쌓는 포트폴리오 레이아웃입니다.
섹션 순서: Hero → About(경력+스킬) → Projects(이미지 슬라이더) → Contact

> 📌 이 파일은 `/project-init`, `/theme-on`, `/sample-page-create`가 참조하는 스타일 정의입니다.

> ⚠️ **아키텍처 원칙**: 컴포넌트는 `import { resume }` 없이 **props**로 데이터를 받습니다.
> `data/resume.ts`는 commands가 읽는 소스이며, 런타임 import 대상이 아닙니다.
> `page.tsx`에서 resume.ts 값을 하드코딩하여 props로 전달합니다.

## Block 스타일 특징

| 섹션 | 구성 |
|------|------|
| Hero | 전체 화면 높이, 중앙 정렬, 이름+직함+태그라인 |
| About | 경력 타임라인 (좌) + 스킬 태그 (우) — 2칼럼 grid |
| Projects | 텍스트 (좌) + 이미지 슬라이더 (우) — 프로젝트별 카드 |
| Contact | 이메일/전화 텍스트 직접 노출 + GitHub 링크 |

## 컴포넌트 위치

```
src/components/block/
  hero.tsx
  about.tsx
  projects.tsx    ← 이미지 슬라이더 + 모달 포함
  contact.tsx
```

  hidden: { opacity: 0, y: 24 },
  visible: (i: number) => ({
    opacity: 1,
    y: 0,
    transition: { duration: 0.5, delay: i * 0.1, ease: 'easeOut' },

interface ModalState { src: string; alt: string; }

interface ProjectUrl { label: string; href: string; }
interface ProjectCase {
  id: string; title: string; company?: string; role?: string; description?: string;
  execution: string; tools?: string; year?: string;
  urls?: ProjectUrl[]; images: string[];
}
interface ProjectsSectionProps { intro: string; cases: ProjectCase[]; }

export function ProjectsSection({ intro, cases }: ProjectsSectionProps) {
  const [modal, setModal] = useState<ModalState | null>(null);
  const [slideIndices, setSlideIndices] = useState<Record<string, number>>({});

  const closeModal = useCallback(() => setModal(null), []);
  const getIdx = (id: string) => slideIndices[id] ?? 0;

  const goSlide = useCallback((id: string, total: number, dir: 1 | -1) => {
    setSlideIndices((prev) => ({
      ...prev,
      [id]: ((prev[id] ?? 0) + dir + total) % total,
    }));
  }, []);

  useEffect(() => {
    if (!modal) return;
    const onKey = (e: KeyboardEvent) => { if (e.key === 'Escape') closeModal(); };
    window.addEventListener('keydown', onKey);
    document.body.style.overflow = 'hidden';
    return () => {
      window.removeEventListener('keydown', onKey);
      document.body.style.overflow = '';
    };
  }, [modal, closeModal]);

  return (
    <>
      <Section id="projects">
        <Container>
          <SectionTitle>프로젝트</SectionTitle>
          <Intro>{intro}</Intro>
          {cases.map((c, i) => {
            const images = c.images ?? [];

              <CaseCard

                          <span className="material-symbols-outlined" style={{ fontSize: '14px' }}>open_in_new</span>
                          {u.label}
                        </CardLink>
                      ))}
                    </CardLinks>
                  )}
                </CardBody>

                {images.length === 1 && (
                  <ImageSide>
                    <ImageWrapper onClick={() => setModal({ src: images[0], alt: c.title })}>
                      <Image src={images[0]} alt={c.title} fill style={{ objectFit: 'cover' }} sizes="(max-width: 768px) 100vw, 45vw" />
                    </ImageWrapper>
                  </ImageSide>
                )}

                {images.length >= 2 && (
                  <ImageSide>
                    <SliderContainer>
                      <SliderTrack $index={getIdx(c.id)}>
                        {images.map((src, idx) => (
                          <Slide key={src} onClick={() => setModal({ src, alt: `${c.title} ${idx + 1}` })}>
                            <Image src={src} alt={`${c.title} ${idx + 1}`} fill style={{ objectFit: 'cover' }} sizes="(max-width: 768px) 100vw, 45vw" />
                          </Slide>
                        ))}
                      </SliderTrack>
                      <SlideNavButton data-dir="prev" onClick={(e) => { e.stopPropagation(); goSlide(c.id, images.length, -1); }} aria-label="이전">‹</SlideNavButton>
                      <SlideNavButton data-dir="next" onClick={(e) => { e.stopPropagation(); goSlide(c.id, images.length, 1); }} aria-label="다음">›</SlideNavButton>
                      <SliderDots>
                        {images.map((_, idx) => (
                          <SliderDot key={idx} $active={getIdx(c.id) === idx} onClick={(e) => { e.stopPropagation(); setSlideIndices((prev) => ({ ...prev, [c.id]: idx })); }} />
                        ))}
                      </SliderDots>
                    </SliderContainer>
                  </ImageSide>

## Rules

- 모든 스타일은 `styled-components`, `props.theme.*` 토큰 사용
- 이미지는 `next/image` + `unoptimized` 사용
- 애니메이션은 Framer Motion (`whileInView` + `viewport={{ once: true }}`)
- 섹션 컴포넌트는 `src/components/block/`에만 위치
- 아이콘: Google Fonts Material Symbols (`layout.tsx`에 CDN 추가)

## 이미지 처리 규칙 (projects.tsx)

이미지 없거나 로드 실패 시 엑박 대신 그라디언트 플레이스홀더로 처리:

```
images.length === 0 → ImageSide 미렌더링, 카드 전체 너비 텍스트 레이아웃
images.length > 0   → ImageSide 렌더링
  └ onError 발생 시 → ImgFallback 그라디언트 표시 (모달 열기 방지)
```
