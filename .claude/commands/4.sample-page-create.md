🟢 **Phase 1 — 샘플 페이지 생성** (Phase 1 마지막 단계)

포트폴리오 샘플페이지를 `styled-components`로 생성합니다.

⚠️ **사전 조건**: `/project-init` → 테마 커맨드 → `/content-setup` 순서대로 먼저 실행되어야 합니다.

## Step 0: 테마 선택 (필수)

**`$ARGUMENTS`가 없는 경우**, 아래 항목을 확인하고 사용자에게 테마를 선택하도록 요청하세요:

```bash
# 설치된 테마 확인
ls jione-portfolio/src/styles/themes/
ls jione-portfolio/src/app/
```

확인 후 사용자에게 아래 옵션을 제시하세요:

```
어떤 테마로 페이지를 생성할까요?

  1) Toss    → /toss   라우트에 생성 (src/app/toss/page.tsx)
               ↳ 깔끔한 Blue 계열, 둥근 카드 — fintech 스타일
  2) Minimal → /minimal 라우트에 생성 (src/app/minimal/page.tsx)
               ↳ 흑백, 각진 레이아웃 — 타이포그래피 중심
  3) Dark    → /dark   라우트에 생성 (src/app/dark/page.tsx)
               ↳ 다크 배경, 퍼플 글로우 — 개발자/테크 스타일
  4) Kakao   → /kakao  라우트에 생성 (src/app/kakao/page.tsx)
               ↳ Yellow 강조, 카드 그리드, 다크 네비 — 카카오 기업 사이트 스타일
  5) Naver   → /naver  라우트에 생성 (src/app/naver/page.tsx)
               ↳ 다크 히어로 + 화이트 카드 교차, 네이버 그린 — 글로벌 테크 기업 스타일

※ 해당 테마가 아직 설치되지 않았다면, 먼저 /toss-theme-on, /theme-minimal, /theme-dark, /theme-kakao, /theme-naver 중 해당 커맨드를 실행하세요.
```

사용자 선택에 따라 아래 변수를 결정하세요:

| 선택 | TARGET_ROUTE | THEME_FILE | THEME_IMPORT |
|------|-------------|-----------|--------------|
| 1) Toss | `toss` | `src/styles/themes/toss.ts` | `tossTheme` |
| 2) Minimal | `minimal` | `src/styles/themes/minimal.ts` | `minimalTheme` |
| 3) Dark | `dark` | `src/styles/themes/dark.ts` | `darkTheme` |
| 4) Kakao | `kakao` | `src/styles/themes/kakao.ts` | `kakaoTheme` |
| 5) Naver | `naver` | `src/styles/themes/naver.ts` | `naverTheme` |

이후 모든 Step에서 `{TARGET_ROUTE}` 는 선택된 값으로 대체합니다.
(예: Toss 선택 시 `src/app/toss/page.tsx` 에 생성)

## 사전 체크

아래 항목들을 확인하세요:

#### 2-3. Project Section

`resume.Project.cases`의 프로젝트를 카드 형태로 렌더링합니다.

- 레이아웃: 왼쪽 텍스트 + 오른쪽 이미지 (desktop), 세로 쌓기 (mobile)
- 이미지 1장: 단일 이미지 표시
- 이미지 2장 이상: 슬라이드 형식 (←/→ 버튼 + 하단 도트 인디케이터)
- 이미지 클릭 시 모달 팝업으로 크게 보기 (ESC·배경 클릭으로 닫기)

```typescript
'use client';

import Image from 'next/image';
import { useState, useEffect, useCallback } from 'react';
import styled, { keyframes } from 'styled-components';
import { resume } from '@/data/resume';

const Section = styled.section`
  padding: 5rem 1.5rem;
  background-color: ${(props) => props.theme.colors.muted};
`;

// 이미지 유무에 따라 flex-direction을 전환 (mobile: column, desktop: row)
const CaseCard = styled.div<{ $hasImages: boolean }>`
  background-color: ${(props) => props.theme.colors.card};
  border-radius: ${(props) => props.theme.radius.xl};
  box-shadow: ${(props) => props.theme.shadows.sm};
  overflow: hidden;
  transition: box-shadow 0.2s ease;
  display: flex;
  flex-direction: column;

  @media (min-width: 768px) {
    flex-direction: ${({ $hasImages }) => ($hasImages ? 'row' : 'column')};
  }

  &:hover {
    box-shadow: ${(props) => props.theme.shadows.md};
  }
`;

const CardBody = styled.div`
  padding: 2rem;
  flex: 1;
`;

// 오른쪽 이미지 영역
const ImageSide = styled.div`
  display: flex;
  flex-direction: column;
  border-top: 1px solid ${(props) => props.theme.colors.border};

  @media (min-width: 768px) {
    border-top: none;
    border-left: 1px solid ${(props) => props.theme.colors.border};
    flex: 0 0 45%;
  }
`;

// 단일 이미지 래퍼 — hover 시 🔍 오버레이 표시
const ImageWrapper = styled.div`
  position: relative;
  flex: 1;
  min-height: 200px;
  background-color: ${(props) => props.theme.colors.secondary};
  cursor: zoom-in;

  &:hover::after {
    content: '🔍';
    position: absolute;
    inset: 0;
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: rgba(0, 0, 0, 0.25);
    font-size: 2rem;
  }
`;

// ─── 슬라이더 (이미지 2장 이상) ────────────────────────────────────

// overflow: hidden 으로 현재 슬라이드만 노출
const SliderContainer = styled.div`
  position: relative;
  overflow: hidden;
  flex: 1;
  min-height: 200px;
`;

// translateX 로 슬라이드 이동
const SliderTrack = styled.div<{ $index: number }>`
  display: flex;
  width: 100%;
  height: 100%;
  transform: translateX(${({ $index }) => -$index * 100}%);
  transition: transform 0.35s cubic-bezier(0.16, 1, 0.3, 1);
`;

// 개별 슬라이드 — 클릭 시 모달
const Slide = styled.div`
  position: relative;
  flex: 0 0 100%;
  min-height: 200px;
  background-color: ${(props) => props.theme.colors.secondary};
  cursor: zoom-in;

  &:hover::after {
    content: '🔍';
    position: absolute;
    inset: 0;
    display: flex;
    align-items: center;
    justify-content: center;
    background-color: rgba(0, 0, 0, 0.25);
    font-size: 2rem;
  }
`;

// 이전 / 다음 버튼
const SlideNavButton = styled.button`
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  z-index: 10;
  background-color: rgba(0, 0, 0, 0.45);
  color: #fff;
  border: none;
  border-radius: 50%;
  width: 2rem;
  height: 2rem;
  display: flex;
  align-items: center;
  justify-content: center;
  cursor: pointer;
  font-size: 1.25rem;
  line-height: 1;
  transition: background-color 0.15s ease;

  &:hover {
    background-color: rgba(0, 0, 0, 0.7);
  }
  &[data-dir='prev'] { left: 0.5rem; }
  &[data-dir='next'] { right: 0.5rem; }
`;

// 하단 도트 인디케이터 컨테이너
const SliderDots = styled.div`
  position: absolute;
  bottom: 0.5rem;
  left: 50%;
  transform: translateX(-50%);
  display: flex;
  gap: 0.375rem;
  z-index: 10;
`;

// 개별 도트
const SliderDot = styled.button<{ $active: boolean }>`
  width: 6px;
  height: 6px;
  border-radius: 50%;
  border: none;
  padding: 0;
  cursor: pointer;
  background-color: ${({ $active }) =>
    $active ? 'rgba(255,255,255,1)' : 'rgba(255,255,255,0.45)'};
  transition: background-color 0.15s ease;
`;

// ─── 모달 애니메이션 ────────────────────────────────────────────────

const fadeIn = keyframes`
  from { opacity: 0; }
  to   { opacity: 1; }
`;

const scaleIn = keyframes`
  from { opacity: 0; transform: scale(0.92); }
  to   { opacity: 1; transform: scale(1); }
`;

// ─── 모달 ──────────────────────────────────────────────────────────

const ModalOverlay = styled.div`
  position: fixed;
  inset: 0;
  background-color: rgba(0, 0, 0, 0.88);
  display: flex;
  align-items: center;
  justify-content: center;
  z-index: 1000;
  padding: 2rem;
  cursor: zoom-out;
  animation: ${fadeIn} 0.2s ease;
`;

const ModalContent = styled.div`
  position: relative;
  max-width: min(90vw, 960px);
  width: 100%;
  cursor: default;
  animation: ${scaleIn} 0.25s cubic-bezier(0.16, 1, 0.3, 1);
`;

const CloseButton = styled.button`
  position: absolute;
  top: -2.75rem;
  right: 0;
  background: none;
  border: none;
  color: rgba(255, 255, 255, 0.8);
  font-size: 1.75rem;
  line-height: 1;
  padding: 0.25rem 0.5rem;
  cursor: pointer;
  transition: color 0.15s ease;

  &:hover {
    color: #ffffff;
  }
`;

const ModalImageWrapper = styled.div`
  position: relative;
  aspect-ratio: 16 / 9;
  border-radius: 16px;
  overflow: hidden;
  box-shadow: 0 32px 80px rgba(0, 0, 0, 0.6);
`;

// ─── 컴포넌트 ──────────────────────────────────────────────────────

interface ModalState {
  src: string;
  alt: string;
}

export function ProjectSection() {
  const { Project } = resume;
  const [modal, setModal] = useState<ModalState | null>(null);
  // 케이스별 현재 슬라이드 인덱스 { [caseId]: number }
  const [slideIndices, setSlideIndices] = useState<Record<string, number>>({});

  const closeModal = useCallback(() => setModal(null), []);

  const getIdx = (id: string) => slideIndices[id] ?? 0;

  const goSlide = useCallback((id: string, total: number, dir: 1 | -1) => {
    setSlideIndices((prev) => ({
      ...prev,
      [id]: ((prev[id] ?? 0) + dir + total) % total,
    }));
  }, []);

  // ESC 키로 닫기 + body 스크롤 잠금
  useEffect(() => {
    if (!modal) return;
    const onKey = (e: KeyboardEvent) => {
      if (e.key === 'Escape') closeModal();
    };
    window.addEventListener('keydown', onKey);
    document.body.style.overflow = 'hidden';
    return () => {
      window.removeEventListener('keydown', onKey);
      document.body.style.overflow = '';
    };
  }, [modal, closeModal]);

  return (
    <>
      <Section id="Project">
        {Project.cases.map((c) => {
          const images = ('images' in c ? c.images : []) as readonly string[];
          return (
            <CaseCard key={c.id} $hasImages={images.length > 0}>
              {/* 왼쪽: 텍스트 */}
              <CardBody>
                {/* 텍스트 콘텐츠 */}
              </CardBody>

              {/* 오른쪽: 이미지 1장 — 단일 표시 */}
              {images.length === 1 && (
                <ImageSide>
                  <ImageWrapper
                    onClick={() => setModal({ src: images[0], alt: c.title })}
                    role="button"
                    aria-label={`${c.title} 이미지 크게 보기`}
                  >
                    <Image
                      src={images[0]}
                      alt={c.title}
                      fill
                      style={{ objectFit: 'cover' }}
                      sizes="(max-width: 768px) 100vw, 45vw"
                    />
                  </ImageWrapper>
                </ImageSide>
              )}

              {/* 오른쪽: 이미지 2장 이상 — 슬라이드 형식 */}
              {images.length >= 2 && (
                <ImageSide>
                  <SliderContainer>
                    <SliderTrack $index={getIdx(c.id)}>
                      {images.map((src, i) => (
                        <Slide
                          key={src}
                          onClick={() =>
                            setModal({ src, alt: `${c.title} ${i + 1}` })
                          }
                          role="button"
                          aria-label={`${c.title} 이미지 ${i + 1} 크게 보기`}
                        >
                          <Image
                            src={src}
                            alt={`${c.title} ${i + 1}`}
                            fill
                            style={{ objectFit: 'cover' }}
                            sizes="(max-width: 768px) 100vw, 45vw"
                          />
                        </Slide>
                      ))}
                    </SliderTrack>

                    {/* 이전 버튼 */}
                    <SlideNavButton
                      data-dir="prev"
                      onClick={(e) => {
                        e.stopPropagation();
                        goSlide(c.id, images.length, -1);
                      }}
                      aria-label="이전 이미지"
                    >
                      ‹
                    </SlideNavButton>

                    {/* 다음 버튼 */}
                    <SlideNavButton
                      data-dir="next"
                      onClick={(e) => {
                        e.stopPropagation();
                        goSlide(c.id, images.length, 1);
                      }}
                      aria-label="다음 이미지"
                    >
                      ›
                    </SlideNavButton>

                    {/* 하단 도트 */}
                    <SliderDots>
                      {images.map((_, i) => (
                        <SliderDot
                          key={i}
                          $active={getIdx(c.id) === i}
                          onClick={(e) => {
                            e.stopPropagation();
                            setSlideIndices((prev) => ({
                              ...prev,
                              [c.id]: i,
                            }));
                          }}
                          aria-label={`이미지 ${i + 1}로 이동`}
                        />
                      ))}
                    </SliderDots>
                  </SliderContainer>
                </ImageSide>
              )}
            </CaseCard>
          );
        })}
      </Section>

      {/* 이미지 모달 — 배경 클릭 또는 ESC로 닫기 */}
      {modal && (
        <ModalOverlay onClick={closeModal}>
          <ModalContent onClick={(e) => e.stopPropagation()}>
            <CloseButton onClick={closeModal} aria-label="닫기">×</CloseButton>
            <ModalImageWrapper>
              <Image
                src={modal.src}
                alt={modal.alt}
                fill
                style={{ objectFit: 'contain' }}
                sizes="min(90vw, 960px)"
                priority
              />
            </ModalImageWrapper>
          </ModalContent>
        </ModalOverlay>
      )}
    </>
  );
}
```

## Rules

- **스타일 관리**: 모든 스타일은 `styled-components` 사용
- **테마 일관성**: `src/styles/theme.ts`의 토큰만 사용
- **콘텐츠 보존**: 기존 프로젝트 데이터는 수정하지 않음 (읽기만 수행)
- **반응형**: mobile-first 방식으로 확실하게 개발 (`@media` 쿼리 필수)
- **성능**: 이미지는 `next/image`로 최적화
- **접근성**: Semantic HTML 사용, 적절한 heading 계층 유지
- **빌드**: 생성 후 반드시 `npm run build` 실행하여 에러 확인
- **아이콘 및 커서 리소스**: Google Fonts 아이콘 폰트(예: Material Symbols/Icons) 사용. `layout.tsx` `<head>`에 CDN 링크를 추가한다.
  ```html
  <link
    href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200"
    rel="stylesheet"
  />
  ```
  사용법: `<span className="material-symbols-outlined">icon_name</span>` 혹은 필요 시 커서 스타일에 동일한 폰트에서 글리프를 참조하여 커서 이미지를 생성한다.
- **이미지 커서**: `cursor: zoom-in` 사용 금지. 이미지 hover 효과는 subtle한 dim overlay(`rgba(0,0,0,0.08)`)로만 표현
- **애니메이션**: Framer Motion 사용 (이미 설치됨)
  - `motion.div` + `whileInView` + `viewport={{ once: true }}`로 스크롤 진입 시 fade-up
  - variants는 반드시 `Variants` 타입으로 명시 (`import { motion, type Variants } from 'framer-motion'`)
  - ease는 `'easeOut'` 문자열 대신 `Variants` 타입 선언 시 자동 추론됨
  - Hero: `animate` (페이지 로드 시), 나머지 섹션: `whileInView` (스크롤 진입 시)
  - 카드 목록은 `custom` 인덱스로 stagger delay 적용
