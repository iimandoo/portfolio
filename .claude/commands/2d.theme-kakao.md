🟢 **Phase 1 — 디자인 시스템 · Kakao 테마**

> 📌 공용 인프라(`provider.tsx`, `styled.d.ts`, `globals.css`), 공통 Rules, 빌드 확인, 다음 단계는 **`2.theme-on.md`** 참조

## Kakao 테마 특징

| 항목 | 값 | 비고 |
|------|----|------|
| 배경 | `#FFFFFF` | 순백 배경 |
| 전경 | `#1A1A1A` | 거의 검정 |
| 강조색 | `#FEE500` | 카카오 옐로우 |
| 강조 위 텍스트 | `#1A1A1A` | Yellow는 어두운 텍스트 필요 |
| 카드 radius | `16px` | 적당히 둥글고 모던한 카드 |
| 분위기 | 깔끔한 기업 사이트 | 뱃지·태그 활용, 카드 그리드 |
| 네비 | 다크(`#1A1A1A`) 배경 | 흰 텍스트, Yellow 활성 상태 |

## Kakao 전용 Rules

- **Yellow on Yellow**: `primary` (#FEE500) 배경에는 반드시 `primaryForeground` (#1A1A1A) 사용

## 생성할 파일

| 파일 | 역할 |
|------|------|
| `src/styles/themes/kakao.ts` | Kakao 디자인 토큰 |
| `src/app/kakao/layout.tsx` | Kakao ThemeProvider 래퍼 |

## Steps

### Step 1: `src/styles/themes/kakao.ts` 생성

```typescript
export const kakaoTheme = {
  colors: {
    background: '#FFFFFF',
    foreground: '#1A1A1A',
    primary: '#FEE500',            // 카카오 옐로우 (또는 $ARGUMENTS로 커스텀)
    primaryForeground: '#1A1A1A',  // 노란 배경 위 검정 텍스트
    secondary: '#F7F7F7',
    secondaryForeground: '#3C3C3C',
    muted: '#F0F0F0',
    mutedForeground: '#767676',
    border: '#E8E8E8',
    card: '#FFFFFF',
    cardForeground: '#1A1A1A',
    destructive: '#F03E3E',
    // Kakao 전용 추가 색상
    nav: '#1A1A1A',               // 상단 네비 배경
    navForeground: '#FFFFFF',      // 네비 텍스트
    badge: '#FEE500',              // 카테고리 뱃지 배경
    badgeForeground: '#1A1A1A',    // 뱃지 텍스트
  },

  radius: {
    xs: '0.25rem',
    sm: '0.5rem',
    md: '0.75rem',
    lg: '1rem',
    xl: '1.25rem',   // 16px — 카카오 카드 특유의 둥근 모서리
  },

  shadows: {
    sm: '0 1px 4px rgba(0,0,0,0.05)',
    md: '0 4px 16px rgba(0,0,0,0.08)',
    lg: '0 8px 24px rgba(0,0,0,0.10)',
  },

  fonts: {
    primary: "'Pretendard Variable', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif",
  },

  typography: {
    h1: { fontSize: '2.5rem', fontWeight: 700, letterSpacing: '-0.02em' },
    h2: { fontSize: '1.75rem', fontWeight: 700, letterSpacing: '-0.01em' },
    body: { fontSize: '1rem', fontWeight: 400, lineHeight: 1.6 },
    small: { fontSize: '0.875rem', fontWeight: 400 },
  },
} as const;

export type KakaoTheme = typeof kakaoTheme;
```

### Step 2: `src/app/kakao/layout.tsx` 생성

```typescript
'use client';

import { StyleProvider } from '@/styles/provider';
import { kakaoTheme } from '@/styles/themes/kakao';

export default function KakaoLayout({ children }: { children: React.ReactNode }) {
  return <StyleProvider theme={kakaoTheme}>{children}</StyleProvider>;
}
```

## 섹션 컴포넌트 스타일 가이드 (Kakao 테마)

`/sample-page-create` 실행 시 참고할 Kakao 특화 스타일 지침:

**레이아웃 원칙:**
- 상단 네비게이션: 다크(`#1A1A1A`) 배경, 흰 텍스트, 로고 왼쪽 정렬
- 섹션 제목 옆에 Yellow 배지 또는 Yellow underline 강조
- 카드 그리드: 2~3열, 이미지 위 카테고리 배지
- 섹션 간 구분은 배경색 교체가 아닌 여백으로 처리
