🟢 **Phase 1 — 디자인 스타일 · Corporate**

Corporate 스타일은 카카오, 네이버 등 기업 사이트에서 볼 수 있는 레이아웃입니다.
상단 스티키 GNB + 대형 Hero + 카드 슬라이더(프로젝트) + 경력/스킬 + Contact Footer

> 📌 이 파일은 `/project-init`, `/theme-on`, `/sample-page-create`가 참조하는 스타일 정의입니다.

> ⚠️ **아키텍처 원칙**: 컴포넌트는 `import { resume }` 없이 **props**로 데이터를 받습니다.
> `data/resume.ts`는 commands가 읽는 소스이며, 런타임 import 대상이 아닙니다.
> `page.tsx`에서 resume.ts 값을 하드코딩하여 props로 전달합니다.

## Corporate 스타일 특징

| 섹션 | 구성 |
|------|------|
| GNB | sticky 상단 네비바: 로고(좌) + 메뉴(중) + 유틸 아이콘(우) |
| Hero | 대형 타이포 제목 + 설명 (full-width, 좌정렬) |
| CardSlider | 수평 스크롤 카드 — 뱃지 + 날짜 + 제목 + 태그 |
| Career | 경력 타임라인 2열 그리드 |
| Skills | 스킬 카테고리별 태그 그리드 |
| Contact | 푸터 스타일 contact (배경 어둡게) |

## 컴포넌트 위치

```
src/components/corporate/
  gnb.tsx          ← 스티키 네비게이션 바
  hero.tsx         ← 대형 제목 + 설명 섹션
  card-slider.tsx  ← 수평 슬라이딩 프로젝트 카드
  career.tsx       ← 경력 + 스킬 섹션
  contact.tsx      ← 푸터 스타일 연락처
```

          <CareerGrid>
            {experiences.map((exp, i) => (
              <CareerCard
                key={exp.id}
                variants={itemVariants}
                initial="hidden"
                whileInView="visible"
                viewport={{ once: true }}
                custom={i}
              >
                <Company>{exp.company}</Company>
                <Position>{exp.position}</Position>
                <Period>{exp.period.start} ~ {exp.period.end === 'present' ? '현재' : exp.period.end}</Period>
                <Description>{exp.description}</Description>
              </CareerCard>
            ))}
          </CareerGrid>
        </Container>
      </CareerSection>

      {/* 스킬 — 독립 row */}
      <SkillsSection id="skills">
        <Container>
          <BlockTitle>스킬</BlockTitle>
          <SkillsGrid>
            {skillCategories.map((cat, i) => (
              <SkillBlock
                key={cat.name}
                as={motion.div}
                variants={itemVariants}
                initial="hidden"
                whileInView="visible"
                viewport={{ once: true }}
                custom={i}
              >
                <CategoryLabel>{cat.name}</CategoryLabel>
                <TagRow>
                  {cat.skills.map((skill) => (
                    <SkillTag key={skill.name} $level={skill.level}>{skill.name}</SkillTag>
                  ))}
                </TagRow>
              </SkillBlock>
            ))}
          </SkillsGrid>
        </Container>
      </SkillsSection>
    </>
  );
}

## Rules

- 모든 스타일은 `styled-components`, `props.theme.*` 토큰 사용

- `badge`/`badgeForeground` 토큰 없으면 `primary`/`primaryForeground`로 fallback
- 애니메이션은 Framer Motion (`whileInView` + `viewport={{ once: true }}`)
