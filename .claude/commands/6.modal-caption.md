# 🖼 Modal Image Slider + Caption

block / corporate 프로젝트 섹션 모달에 다음 기능을 추가합니다:

1. **이미지 슬라이더** — 모달 안에서 같은 프로젝트의 이미지를 이전/다음으로 탐색
2. **키보드 지원** — `←` / `→` 화살표 키로 이동, `Esc` 닫기
3. **도트 인디케이터** — 현재 위치 표시 (이미지 2장 이상일 때)
4. **파일명 캡션** — 모달 하단에 이미지 파일명 표시 (확장자·경로 제거)

> 예) `/projects/AI마타수학2026_선생님 홈화면_1.png` → `AI마타수학2026_선생님 홈화면_1`

| `$ARGUMENTS` | 수정 파일 |
|---|---|
| `all` (기본값) | block + corporate 동시 적용 |
| `block` | `src/components/block/projects.tsx` |
| `corporate` | `src/components/corporate/card-slider.tsx` |

### Corporate 적용 시 추가 주의사항

- 기존 `PROJECT_IMAGES` 맵과 `getImgSrc` 헬퍼가 남아있다면 **삭제**하고 `cases` prop의 `c.images` 배열 직접 사용
- `ProjectImageSlider` 서브컴포넌트에 `onOpen: (index: number) => void` prop 추가
- 이미지 경로는 이미 `/projects/...` 형태이므로 `encodeURIComponent` 불필요
- `CardSlider` 상위 컴포넌트에서 modal state 관리 후 `ProjectImageSlider`에 콜백 전달

## Steps

### 1. `ModalState` 인터페이스 변경

기존 `{ src, alt }` → 이미지 배열 + 현재 인덱스 구조로 교체:

```typescript
interface ModalState {
  images: readonly string[];
  index: number;
  title: string;
}
```

### 2. 새 styled-components 추가

`ModalImageWrapper` 정의 아래에 삽입합니다:

const ModalSlider = styled.div`

### 3. `goModal` 콜백 추가

`goSlide` 아래에 추가합니다:

```typescript
const goModal = useCallback((dir: 1 | -1) => {
  setModal((prev) => {
    if (!prev) return prev;
    return { ...prev, index: (prev.index + dir + prev.images.length) % prev.images.length };
  });
}, []);
```

### 4. `useEffect` 키보드 핸들러 업데이트

`ArrowLeft` / `ArrowRight` 지원 추가:

```typescript
useEffect(() => {
  if (!modal) return;
  const onKey = (e: KeyboardEvent) => {
    if (e.key === 'Escape') closeModal();
    if (e.key === 'ArrowLeft') goModal(-1);
    if (e.key === 'ArrowRight') goModal(1);
  };
  window.addEventListener('keydown', onKey);
  document.body.style.overflow = 'hidden';
  return () => {
    window.removeEventListener('keydown', onKey);
    document.body.style.overflow = '';
  };
}, [modal, closeModal, goModal]);
```

### 5. 카드의 `setModal` 호출 교체

**단일 이미지 클릭:**
```tsx
onClick={() => !hasErr(`${c.id}_0`) && setModal({ images, index: 0, title: c.title })}
```

**슬라이더 이미지 클릭:**
```tsx
onClick={() => !hasErr(`${c.id}_${idx}`) && setModal({ images, index: idx, title: c.title })}
```

### 6. 모달 JSX 교체

```tsx
{modal && (
  <ModalOverlay onClick={closeModal}>
    <ModalContent onClick={(e) => e.stopPropagation()}>
      <CloseButton onClick={closeModal}><span className="material-symbols-outlined">close</span></CloseButton>
      <ModalSlider>
        <ModalImageWrapper>
          <Image
            src={modal.images[modal.index]}
            alt={`${modal.title} ${modal.index + 1}`}
            fill
            style={{ objectFit: 'contain' }}
            sizes="min(90vw, 960px)"
            priority
          />
        </ModalImageWrapper>
        {modal.images.length > 1 && (
          <>
            <ModalNavBtn data-dir="prev" onClick={() => goModal(-1)} aria-label="이전"><span className="material-symbols-outlined">chevron_left</span></ModalNavBtn>
            <ModalNavBtn data-dir="next" onClick={() => goModal(1)} aria-label="다음"><span className="material-symbols-outlined">chevron_right</span></ModalNavBtn>
          </>
        )}
      </ModalSlider>
      {modal.images.length > 1 && (
        <ModalDots>
          {modal.images.map((_, idx) => (
            <ModalDot
              key={idx}
              $active={modal.index === idx}
              onClick={() => setModal((prev) => prev ? { ...prev, index: idx } : prev)}
              aria-label={`${idx + 1}번째 이미지`}
            />
          ))}
        </ModalDots>
      )}
      <ModalCaption>
        {modal.images[modal.index].split('/').pop()?.replace(/\.[^.]+$/, '') ?? modal.title}
      </ModalCaption>
    </ModalContent>
  </ModalOverlay>
)}
```

## Rules

- `ModalNavBtn`은 `ModalSlider` (position: relative) 기준으로 절대 위치
- 이미지 1장인 프로젝트는 nav 버튼·dots 미표시
- 키보드 핸들러는 `goModal` 의존성을 `useEffect` deps 배열에 포함
- 파일명 파싱: `src.split('/').pop()?.replace(/\.[^.]+$/, '') ?? title`
