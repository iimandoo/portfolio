🟢 **Phase 1 — 톤앤매너 · Deep Purple**

> 📌 공용 인프라(`provider.tsx`, `styled.d.ts`, `globals.css`), 공통 Rules, 빌드 확인은 **`2.theme-on.md`** 참조

## Deep Purple 톤 특징

| 항목 | 값 | 비고 |
|------|----|------|
| 배경 | `#FAFAFA` | 살짝 회색 배경 (shadcn 스타일) |
| 전경 | `#0F0B15` | 보라 기운 있는 거의 검정 |
| 강조색 | `#7C3AED` | Deep Purple (Violet-600) |
| 카드 | `#FFFFFF` 순백 | 배경 위에 떠 있는 카드 |
| 카드 radius | `12px` | shadcn/ui 기본 라운딩 |
| 보더 | `#E5E7EB` | 미세한 구분선 |
| 분위기 | 클린 SaaS 대시보드 | 미니멀, 카드 기반, 보라 포인트 |
| 그림자 | 거의 없음 | 보더 기반 구분, 초경량 그림자 |

## Deep Purple 전용 Rules

- **보라색 강조**: 버튼, 차트 포인트, 캘린더 활성일, 뱃지에만 `primary` 사용
- **카드 구분**: 배경색이 회색(`#FAFAFA`)이므로 카드는 순백(`#FFFFFF`)으로 대비
- **보더 기반 레이아웃**: 그림자보다 `border` 1px로 카드 영역 구분
- **인터랙션**: hover 시 `primary` 배경 + 흰 텍스트 전환

## 스타일 가이드 힌트

**레이아웃 원칙:**
- 배경은 `#FAFAFA` 회색, 카드는 `#FFFFFF` 순백 — 카드가 떠 있는 느낌
- 카드에 `border: 1px solid` + 최소 그림자 조합
- 보라색은 CTA 버튼, 뱃지 텍스트, 차트 포인트, 활성 상태에만 사용
- 여백 넉넉하게 — 카드 간 `gap: 1rem ~ 1.5rem`

**컴포넌트 예시:**

```typescript
// 카드 — 순백 + 보더 기반 (그림자 최소)
const Card = styled.div`
  background-color: ${(props) => props.theme.colors.card};
  border: 1px solid ${(props) => props.theme.colors.border};
  border-radius: ${(props) => props.theme.radius.md};
  box-shadow: ${(props) => props.theme.shadows.sm};

// 보라 뱃지 — 연보라 배경 + 보라 텍스트

  border-radius: ${(props) => props.theme.radius.sm};

// CTA 버튼 — 보라 배경 + 흰 텍스트
const PrimaryButton = styled.button`

  color: ${(props) => props.theme.colors.primaryForeground};
  border: none;
  border-radius: ${(props) => props.theme.radius.sm};
  padding: 0.5rem 1rem;

  font-weight: 500;
  cursor: pointer;

  &:hover {
    opacity: 0.9;
  }
`;

// 스킬 태그 — 연보라 pill
const SkillTag = styled.span<{ $level: string }>`
  padding: 0.25rem 0.75rem;

  font-weight: 500;

  color: ${(props) =>

// 메트릭 카드 — 큰 숫자 + 소수점 강조
const MetricValue = styled.span`

  font-weight: 700;
  color: ${(props) => props.theme.colors.foreground};
  letter-spacing: -0.025em;
`;

const MetricLabel = styled.span`
  font-size: 0.75rem;
  color: ${(props) => props.theme.colors.mutedForeground};
  text-transform: uppercase;
  letter-spacing: 0.05em;
`;
```

**shadcn 스타일 특징 (참고):**
- `font-size: 0.875rem` (14px) 기본 body — 대시보드에 최적
- `letter-spacing: -0.025em` 타이틀 — 타이트하고 모던한 느낌
- `border-radius: 0.75rem` (12px) 카드 — 적당히 둥글되 과하지 않음
- 그림자 대신 보더로 구분 — 플랫하고 클린한 인상
- 보라색 농도 조절: `#7C3AED` (100%) → `#DDD6FE` (30%) → `#F3F0FF` (10%)
