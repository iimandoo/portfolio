🟢 **Phase 1 — 디자인 시스템 · Naver 테마**

`/naver` 라우트에 네이버 기업 사이트 스타일 디자인 토큰을 적용합니다.

⚠️ **사전 조건**: `/project-init` → `/toss-theme-on` 순서로 먼저 실행되어야 합니다.
(`provider.tsx`, `styled.d.ts`는 `/toss-theme-on`이 생성합니다)

## Naver 테마 특징

> **디자인 레퍼런스**: navercorp.com (뉴타입 에이전시 리뉴얼, 2024)
> 키워드: "글로벌, 연결, 확장 — Global Tech Company 3.0"

| 항목 | 값 | 비고 |
|------|----|------|
| 기본 배경 | `#FFFFFF` | 카드·콘텐츠 섹션 흰 배경 |
| Hero 배경 | `#0D0D0D` | 다크 풀스크린 히어로 (하드코딩) |
| 강조색 | `#03C75A` | 네이버 브랜드 그린 |
| 보조 그린 | `#00C73C` | hover 시 진한 그린 |
| 카드 radius | `12–16px` | 모던하되 날카롭지 않은 모서리 |
| 분위기 | 다크↔라이트 교차 | 섹션마다 배경 전환으로 리듬감 |
| 타이포 | Light weight 대형 | 글로벌 테크 기업 특유의 와이드 제목 |
| 모션 | 스크롤 진입 fade-up | Framer Motion whileInView |

## 생성할 파일

| 파일 | 역할 |
|------|------|
| `src/styles/themes/naver.ts` | Naver 디자인 토큰 |
| `src/app/naver/layout.tsx` | Naver ThemeProvider 래퍼 |

### Step 1: `src/styles/themes/naver.ts` 생성

```typescript
    primaryForeground: '#FFFFFF',
    secondary: '#F4F4F4',
    secondaryForeground: '#333333',
    muted: '#F8F8F8',
    mutedForeground: '#888888',
    border: '#E8E8E8',
    card: '#FFFFFF',

  },

  radius: {
    xs: '0.25rem',
    sm: '0.5rem',
    md: '0.75rem',
    lg: '1rem',
    xl: '1.25rem',   // 20px — 네이버 카드 특유의 둥근 느낌
  },

  shadows: {
    sm: '0 1px 4px rgba(0,0,0,0.06)',
    md: '0 4px 20px rgba(0,0,0,0.08)',
    lg: '0 8px 40px rgba(0,0,0,0.12)',
  },

  fonts: {
    primary: "'Pretendard Variable', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif",
  },

  typography: {
    h1: { fontSize: '3.5rem', fontWeight: 300, letterSpacing: '-0.03em' },  // Light weight 대형 — 글로벌 테크 느낌
    h2: { fontSize: '2.25rem', fontWeight: 400, letterSpacing: '-0.02em' },
    body: { fontSize: '1rem', fontWeight: 400, lineHeight: 1.65 },
    small: { fontSize: '0.875rem', fontWeight: 400 },
  },
} as const;

export type NaverTheme = typeof naverTheme;
```

### Step 2: `src/app/naver/layout.tsx` 생성

> `src/styles/provider.tsx`는 `/toss-theme-on`이 이미 생성했습니다. 없으면 생성하세요 (내용은 `/toss-theme-on` 참고).

```typescript
'use client';

import { StyleProvider } from '@/styles/provider';
import { naverTheme } from '@/styles/themes/naver';

export default function NaverLayout({ children }: { children: React.ReactNode }) {
  return <StyleProvider theme={naverTheme}>{children}</StyleProvider>;
}
```

**Framer Motion 진입 애니메이션 (Naver 스타일):**

```typescript
// 네이버 기업사이트 특유의 — 아래에서 위로 올라오며 fade in
const fadeUpVariants = {
  hidden: { opacity: 0, y: 40 },
  visible: {
    opacity: 1,
    y: 0,
    transition: { duration: 0.7, ease: [0.16, 1, 0.3, 1] },  // easeOutExpo
  },
};

// 섹션 컨테이너에 적용
<motion.div
  variants={fadeUpVariants}
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true, margin: '-80px' }}
>
  {/* 섹션 내용 */}
</motion.div>

// Hero 텍스트는 stagger로 순차 등장
const heroContainerVariants = {
  hidden: {},
  visible: {
    transition: { staggerChildren: 0.12 },
  },

```

## Rules

- **`provider.tsx`, `styled.d.ts`**: 수정하지 않음 — `/toss-theme-on`이 생성한 파일 재사용
- **Hero / Contact 다크 배경**: `#0D0D0D` 하드코딩 (테마 토큰이 아닌 디자인 결정)
- **섹션 컴포넌트**: `src/components/sections/`를 수정하지 않음 — `page.tsx`에서 import
- **빌드 성공**: 반드시 `npm run build` 실행 후 에러 없음 확인
