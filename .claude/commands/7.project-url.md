🟡 **유틸 — 프로젝트 URL 링크 추가**

프로젝트 카드에 외부 링크(서비스 URL, GitHub 등)를 표시하는 기능을 추가합니다.
URL이 있는 카드에만 링크 버튼이 표시되고, 클릭 시 새 탭으로 열립니다.

⚠️ **사전 조건**: `/project-init` + `/content-setup` 이 완료되어 있어야 합니다.

---

## Step 1. resume.ts — `urls` 필드 추가

`data/resume.ts` 파일의 `Project.cases` 배열에서
각 case 객체에 `urls` 필드를 추가합니다 (선택 필드, 없으면 링크 미표시).

**타입 구조:**
```typescript
urls?: readonly { label: string; href: string }[];
```

**예시:**
```typescript
{
  id: 'cool-hanja',
  title: '쿨한자',
  // ... 기존 필드 ...
  urls: [
    { label: '서비스 보기', href: 'https://cool-hanja.vercel.app' },
    { label: 'GitHub', href: 'https://github.com/username/cool-hanja' },
  ],
},
```

- URL이 없는 케이스는 `urls` 필드를 생략하거나 빈 배열로 둡니다.
- `urls` 필드가 없거나 빈 배열이면 카드에 링크 영역이 표시되지 않습니다.

## Step 2. block/projects.tsx — URL 링크 렌더링

`src/components/block/projects.tsx`의 `CardBody` 안, 기존 도구(Tools) 정보 아래에 URL 링크 영역을 추가합니다.

**추가할 styled components:**

```tsx
const CardLinks = styled.div`
  display: flex;
  flex-wrap: wrap;
  gap: 0.5rem;
  margin-top: 0.75rem;
`;

const CardLink = styled.a`
  display: inline-flex;
  align-items: center;
  gap: 0.3rem;
  font-size: 0.8125rem;
  font-weight: 500;
  color: ${(props) => props.theme.colors.primary};
  background-color: ${(props) => props.theme.colors.secondary};
  border: 1px solid ${(props) => props.theme.colors.border};
  border-radius: ${(props) => props.theme.radius.md};
  padding: 0.25rem 0.625rem;
  text-decoration: none;
  transition: background-color 0.15s ease, border-color 0.15s ease;

  &:hover {
    background-color: ${(props) => props.theme.colors.muted};
    border-color: ${(props) => props.theme.colors.primary};
  }
`;
```

**CardBody 내 렌더링 위치 (tools 정보 바로 아래):**

```tsx
{'urls' in c && Array.isArray(c.urls) && c.urls.length > 0 && (
  <CardLinks>
    {(c.urls as readonly { label: string; href: string }[]).map((u) => (
      <CardLink key={u.href} href={u.href} target="_blank" rel="noopener noreferrer">
        <span className="material-symbols-outlined" style={{ fontSize: '14px' }}>open_in_new</span>
        {u.label}
      </CardLink>
    ))}
  </CardLinks>
)}
```

## Rules

- `urls` 필드는 선택(optional) — 없거나 빈 배열이면 UI에 아무것도 표시하지 않음
- 링크는 반드시 `target="_blank" rel="noopener noreferrer"` 로 새 탭 오픈
- URL 개수 제한 없음 (다수 URL 가능)
- `CardLinks` 컴포넌트는 `flex-wrap: wrap` 으로 여러 URL 시 줄바꿈 처리
- styled-components 이름 충돌 없도록 파일별 독립 정의
