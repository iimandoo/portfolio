### 3. 공통 스타일 가이드라인 (styled-components)

**타이포그래피 믹스인:**

```typescript
// src/styles/typography.ts
import styled from "styled-components";

export const H1 = styled.h1`
  font-size: 2.25rem;
  font-weight: 700;
  letter-spacing: -0.02em;
`;

export const H2 = styled.h2`
  font-size: 1.875rem;
  font-weight: 700;
  letter-spacing: -0.02em;
`;

export const Body = styled.p`
  font-size: 1rem;
  font-weight: 400;
  line-height: 1.6;
`;

export const Small = styled.small`
  font-size: 0.875rem;
  font-weight: 400;
  color: ${(props) => props.theme.colors.mutedForeground};
`;
```

**스페이싱:**

```typescript
/* 섹션 상하 */
padding: 3rem 1.5rem; /* mobile */
padding: 5rem 2rem;   /* tablet */
padding: 7rem 3rem;   /* desktop */

/* 컨테이너 */
max-width: 64rem;
margin: 0 auto;

/* 요소 간격 */
gap: 1.5rem;
```

**컬러 팔레트 (styled-components theme):**

```typescript
${props => props.theme.colors.primary}           // #3182F6
${props => props.theme.colors.background}        // #FFFFFF
${props => props.theme.colors.muted}             // #F9FAFB
${props => props.theme.colors.foreground}        // #1B1C1F
${props => props.theme.colors.mutedForeground}   // #8B95A1
```

**쉐도우:**

```typescript
${props => props.theme.shadows.sm}   // 0 1px 4px rgba(0,0,0,0.06)
${props => props.theme.shadows.md}   // 0 4px 16px rgba(0,0,0,0.08)
${props => props.theme.shadows.lg}   // 0 8px 32px rgba(0,0,0,0.10)
```

## Rules

- **스타일 관리**: 모든 스타일은 `styled-components` 사용
- **테마 일관성**: `src/styles/theme.ts`의 토큰만 사용
- **콘텐츠 보존**: 기존 프로젝트 데이터는 수정하지 않음 (읽기만 수행)
- **반응형**: mobile-first 방식으로 확실하게 개발 (`@media` 쿼리 필수)
- **성능**: 이미지는 `next/image`로 최적화
- **접근성**: Semantic HTML 사용, 적절한 heading 계층 유지
- **빌드**: 생성 후 반드시 `npm run build` 실행하여 에러 확인
